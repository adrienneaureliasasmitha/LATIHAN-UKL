# LATIHAN-UKL SOAL 1 NOMOR 1 (menghitung biaya total pengiriman berdasarkan berat paket jarak tempuh serta volume barang yang diketahui)

-Berat paket (dalam kilogram) -Jarak pengiriman (dalam kilometer) -Volume paket (dalam cm³, dihitung dari panjang × lebar × tinggi)

Biaya pengiriman ditentukan oleh: -Jika jarak ≤ 10 km, maka biaya per kilogram = Rp 4.250 -Jika jarak > 10 km, maka biaya per kilogram = Rp 6.000 -Jika volume > 100 cm³, maka ditambah biaya Rp 50.000
